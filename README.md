# qiskit-champions

The main data analysis file parameters_data_analysis_classic_and_quantum.ipynb requires the file brain_signatures_8d.csv to be downloaded in order to run. The file used to generate brain_signatures_8d.csv, pca-8d.ipynb, requires mounting to the Oasis Keggle in order to function and can take a long time, and so brain_signatures_8d.csv has been preemptively stored for ease of use of parameters_data_analysis_classic_and_quantum.ipynb.
